SnackbarJS

SnackbarJS is a lightweight and jQuery powered plugin made to work in Bootstrap style, just like the Bootstrap tooltips work. Use it is easy as write:

Click me 

var options =  {
content: "Some text", // text of the snackbar
style: "toast", // add a custom class to your snackbar
timeout: 100 // time in milliseconds after the snackbar autohides, 0 is disabled
}
$.snackbar(options);
